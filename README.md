# Launching Jupyter on TACC

A step-by-step guide for running Jupyter Notebook on TACC compute nodes.

**Guide:** [View the tutorial](https://ashleyscruse.github.io/jupyter-on-tacc/)

Covers both methods:
- **Method 1:** SSH + idev + SSH tunnel (command line)
- **Method 2:** TACC Analysis Portal (web browser)

Maintained by Ashley Scruse, Morehouse Supercomputing Facility.
