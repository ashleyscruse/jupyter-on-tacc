# Launching Jupyter on HPC

A step-by-step guide for running Jupyter Notebook on HPC compute nodes.

**Guide:** [View the tutorial](https://ashleyscruse.github.io/jupyter-on-hpc/)

Covers both methods:
- **Method 1:** SSH + idev + SSH tunnel (command line)
- **Method 2:** TACC Analysis Portal (web browser)

Maintained by the Morehouse Supercomputing Facility.
